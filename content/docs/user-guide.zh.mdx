# 欢迎使用 CompareHeights！📏

**CompareHeights** 是一款专业的身高对比工具，允许你在人物、名人、物体和自定义图片之间创建精确的视觉对比。它支持从纳米到公里的极端比例范围和精密测量计算，非常适合教育、专业和个人用途。

## 主要功能

-   🎯 **精确的角色数据库**  
    预置包括名人、物体和生物样本等在内的角色，具有经过验证的身高数据。
-   📐 **精密测量系统**  
    支持从纳米到公里的 9 种不同单位，具备智能单位转换和科学计数法显示。
-   🖼️ **自定义图片上传**  
    可上传并裁剪自定义图片，支持精确身高设定与自动宽高比计算。
-   📊 **交互式可视化**  
    支持拖拽排序、缩放控制、实时测量显示与响应式布局。
-   💾 **专业级导出**  
    可导出高质量 PNG、JPG 或 WebP 图片，适用于演示与分享。
-   📱 **社交媒体优化**  
    内置社交平台分享工具，自动生成优化后的图片输出。

## 如何使用 CompareHeights

### 1. 入门指南

在任意现代浏览器中访问 [compareheights.org](https://compareheights.org)。工具可即时加载，无需注册。

<Image src="/assets/docs/user-guide/home.jpg" alt="Home page" width={800} height={400} />

界面分为三个主要区域：  
- **左侧面板**：角色搜索与分类  
- **中间区域**：交互式对比图  
- **顶部控制区**：单位切换、缩放控制和标题栏

### 2. 添加角色进行对比

#### **方法一：使用预设角色**

1.  **浏览库**
    - **animals**：各种真实或史前动物的身高，从昆虫到恐龙。  
    - **anime**：热门日本动漫角色的身高。  
    - **celebrity**：名人、政治人物、运动员及神话人物的身高。  
    - **fictional_characters**：来自电影、书籍与传说的虚构角色。  
    - **game**：游戏角色和生物的身高（如龙与地下城）。  
    - **generic**：不同年龄、性别和国家的人类平均身高。  
    - **microorganisms**：常见微生物（如细菌、病毒、细胞）的尺寸。  
    - **objects**：自然与人造物体的尺寸，从原子到行星与建筑物。  
    - **plants**：常见植物与树木的高度。  
    - **Upload**：上传你的自定义图片。  

<Image src="/assets/docs/user-guide/library.png" alt="Library" width={800} height={400} />

2.  **搜索角色**  
    使用搜索框输入角色名称即可快速找到目标。

3.  **添加到对比**  
    点击任意角色卡片即可将其添加至中央对比区，并显示其精确高度。

#### **方法二：上传自定义图片**

1.  **进入上传界面**  
    点击“UPLOAD”分类标签打开上传界面。

2.  **上传图片**
    - 拖放图片至上传区域，或  
    - 点击选择文件从设备中上传  
    - 支持格式：JPG、PNG、GIF  

    <Image src="/assets/docs/user-guide/upload.png" alt="Upload" width={800} height={400} />

3.  **裁剪图片**  
    上传后自动打开裁剪工具：  
    - **调整选区**：拖动裁剪区域选择需要的部分  
    - **缩放控制**：使用 +/- 按钮放大或缩小  
    - **重置**：如有需要可重新选择  

4.  **设置身高信息**  
    - **身高输入**：输入真实世界中的身高  
    - **单位选择**：支持 nm、μm、mm、cm、m、km、in、ft、mi  
    - **角色名称**：为角色命名  
    - **自动计算**：宽度会根据裁剪比例自动计算  

5.  **保存角色**  
    点击“Save Character”将自定义图片加入对比图中。

<Image src="/assets/docs/user-guide/crop.png" alt="Crop" width={800} height={400} />

### 3. 管理你的对比图

#### **重新排序角色**

- **拖放排序**：点击并拖拽角色即可调整顺序  
- **实时预览**：拖拽时会实时显示预览效果  
- **最佳布局**：建议按从矮到高排列以获得最佳视觉效果  

<Image src="/assets/docs/user-guide/reorder.png" alt="Reorder" width={800} height={400} />

#### **编辑角色**

1.  **选择角色**：将鼠标悬停在角色上即可看到编辑选项  
2.  **编辑属性**：点击编辑按钮可修改身高、名称等信息  
3.  **删除角色**：点击删除图标（垃圾桶）可移除单个角色  
4.  **清空全部**：使用“Clear All”按钮一次性清空全部角色  

<Image src="/assets/docs/user-guide/edit.png" alt="Edit" width={800} height={400} />
